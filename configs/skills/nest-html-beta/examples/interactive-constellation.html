Interactive constellation

A force-directed concept map you can hover, pin, and zoom into — built for a white page.

Copy the <figure class="concept-graph"> markup, the .concept-graph CSS block, and the mountInteractiveGraph() function from the module below. Feed it { nodes, edges } — that's the only part you change. Everything else (screen-space picking, settle→recentre→build, Line2 gradient edges, halos, projected labels, click-to-pin focus, centre-and-zoom, the dynamic context panel, parallax, the IntersectionObserver mount/dispose lifecycle, and the prefers-reduced-motion path) is load-bearing and already debugged.

Interact: hover a node to preview it · click a node to pin it (it zooms to the middle and a context card opens on the opposite side) · click empty space to zoom back out.

The real takeaway must also live in HTML (a list, a table, prose) above or below the canvas — WebGL can fail (GPU blocklisted, reduced motion, renderer errors), and the scene is an enhancement, never the only copy of a fact.
