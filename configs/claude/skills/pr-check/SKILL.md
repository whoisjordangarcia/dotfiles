# PR Check

Diagnose failed GitHub Actions CI checks on a PR, identify the root cause, and guide fixes.

## When to Use

- CI checks failed on a PR
- User asks "why is the build failing?", "check the PR", "fix CI"
- After pushing and wanting to verify CI status
- When `ci-success` job is failing on branch protection
