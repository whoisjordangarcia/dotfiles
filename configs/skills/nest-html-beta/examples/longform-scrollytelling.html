5Gene research literature — open access

This doc is the counter-example, and it's worth stating: it is almost entirely proportions and concentration. There's no graph to fly through and no flow to trace — so the honest choice is charts and a mermaid pie, not 3D. Forcing a point cloud here would look impressive and read worse.

The clinically-valuable formal guidelines are the least machine-accessible, and the locked content is heavily concentrated in a few publishers. Both are built from JSON data blocks, so updating the numbers never means editing SVG paths.

+Bonus: scrollytelling

One pattern only long-form can do: a pinned visual that advances as you scroll past the prose. The point cloud on the left re-forms at each step — raw noise, normalized onto a sphere, clustered, then settled into an ordered lattice — driven entirely by which paragraph is centered.

scroll to advance
Stage 01 — Ingest

Raw samples arrive as scattered noise

Every run lands as a loose cloud of unlabeled observations — no structure, no shared coordinate system. A variant call and a QC artifact look identical here.

Stage 02 — Normalize

Normalization pulls them onto a sphere

Genome build, units, and terminology are reconciled, so distance starts to mean something. The cloud contracts onto one normalized surface — every point now comparable.

Stage 03 — Cluster

Clustering separates the signal

With a common geometry, similar samples fall together. The surface resolves into colored lobes — cohorts and batches pull apart; the outliers are what a reviewer should see first.

Stage 04 — Deliver

Results settle into a clean lattice

Finally the clusters collapse into a deterministic, reproducible layout that maps one-to-one to the report a clinician reads. Same input, same arrangement, every time.

The takeaway

Graphs get 3D; quantities get charts. Story graphs and request flows earned three.js; the access-tier and publisher splits stayed mermaid pies and bars. Same one-file, on-brand output either way — the only thing that changed between the deck and this article was how the figures mount. Match the visual to the shape of the content, and both formats read.
